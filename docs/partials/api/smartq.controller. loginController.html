loginController
controller in module smartq

Description

Controller responsável pelo login da aplicação. Verifica se o usuário já está logado, faz requisições das informações ao servidor.

Methods

  • doLogin()

    Verifica se o usuário já está logado, se sim redireciona para tela principal da aplicação, se não, redireciona para o login.

  • getServerAgendamentos(id)

    Método que faz requisição ao servidor e salva os dados localmente dos agendamentos do usuário.

    Parameters
    ParamTypeDetails
    idNumber

    Identificador do quadro

  • getServerCircuitos(id)

    Método que faz requisição ao servidor e salva os dados localmente dos Circuitos do usuário.

    Parameters
    ParamTypeDetails
    idNumber

    Identificador do quadro

  • getServerControle(id)

    Método que faz requisição ao servidor e salva os dados localmente dos controles de demanda do usuário.

    Parameters
    ParamTypeDetails
    idNumber

    Identificador do quadro

  • getServerQuadroDetails(id)

    Método que faz requisição ao servidor e salva os dados localmente das informações de um quadro específico do usuário.

    Parameters
    ParamTypeDetails
    idNumber

    Identificador do quadro

  • getServerQuadros()

    Método que faz requisição ao servidor e salva os dados localmente dos quadros do usuário.

    Método utilizado para registrar usuário quando ainda não possui login ativo.
